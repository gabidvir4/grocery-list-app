## How to Use

### Getting Started
1. Open `index.html` in your web browser
2. Start adding items to your grocery list!

### Adding Items
1. Type the item name in the text field
2. Set the quantity (defaults to 1)
3. Click "Add Item" or press Enter

### Managing Items
- **Edit item name**: Click on the item name to edit it inline
- **Change quantity**: 
  - Click the + or - buttons to increment/decrement
  - Click on the quantity number to edit it directly
- **Delete item**: Click the trash can icon (ğŸ—‘ï¸) and confirm

### Data Storage
- Your grocery list is automatically saved to your browser's local storage
